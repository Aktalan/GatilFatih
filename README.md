# Gatil Fatih (One‑Pager)

Statische One‑Pager‑Webseite (HTML/CSS/JS).

## Starten

Option A (einfach): `index.html` doppelklicken.

Option B (lokaler Server, empfohlen):

```bash
python -m http.server 5173
```

Dann im Browser öffnen: `http://localhost:5173`

## Assets

Folgende Dateien werden verwendet:

- `assets/logo.png`
- `assets/hero.png`
- `assets/ref-1.png`
- `assets/ref-2.png`

## Hinweis

Ohne Backend: Das Formular erzeugt eine lokale Zusammenfassung im Browser.
